# Horror Movie Recommender

A fully interactive horror movie recommendation system with rating and review capabilities.

## Features

- **30 Movie Database**: Curated collection of classic and modern horror films
- **Random Selection**: Get 5 random movie recommendations at a time
- **Keyword Filtering**: Click on any keyword tag to filter movies by genre/theme
- **Interactive Star Rating**: Rate each movie from 1-5 stars
- **Review System**: Write and save text reviews for each film
- **Custom Search**: Enter your own keywords to find specific types of horror
- **Persistent Storage**: Your ratings and reviews are automatically saved in browser

## How to Use

1. Open `index.html` in your web browser
2. Browse the 5 randomly selected horror movies
3. Click on keyword tags to filter by themes (supernatural, slasher, psychological, etc.)
4. Use the custom keyword input to search for specific preferences
5. Rate movies by clicking on the stars (1-5)
6. Write reviews in the text areas - they auto-save!
7. Click "Get 5 New Recommendations" to see different movies

## Movie Keywords Include

- Supernatural, Psychological, Slasher, Zombie
- Classic, Modern, Indie, Foreign
- Gore, Suspense, Mystery, Thriller
- Haunted House, Possession, Cult
- And many more!

## Technologies Used

- Pure HTML/CSS/JavaScript (no frameworks required)
- LocalStorage for persistent data
- Responsive design for mobile and desktop
